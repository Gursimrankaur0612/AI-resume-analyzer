# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes against job descriptions, calculates ATS scores, identifies missing skills, generates interview questions, and provides AI-powered resume improvement suggestions.

## 🚀 Live Demo

🌐 https://ai-resume-analyzer-one-red.vercel.app/

## 📂 GitHub Repository

https://github.com/Gursimrankaur0612/AI-resume-analyzer

---

## ✨ Features

- 📄 Resume Upload (PDF)
- 🎯 ATS Score Calculation
- 🧠 AI Resume Analysis
- ✅ Skill Extraction
- ❌ Missing Skill Detection
- 💡 AI Resume Improvement Suggestions
- 🎤 AI Interview Question Generator
- 📊 Dashboard Analytics
- 📜 Resume History
- 🔐 JWT Authentication
- ☁️ Fully Deployed Application
