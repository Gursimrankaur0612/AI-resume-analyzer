# AI Resume Analyzer

An AI-powered Resume Analyzer built using Spring Boot, React, Railway MySQL, Render, Vercel, and Google Gemini AI.

## 🚀 Live Demo

🔗 https://ai-resume-analyzer-one-red.vercel.app

## ✨ Features

- Resume Analysis
- ATS Score
- Resume Improvement Suggestions
- Interview Question Generator
- Dashboard Analytics
- Resume History
- JWT Authentication

## 🛠 Tech Stack

### Frontend
- React
- Bootstrap
- Axios
- Vercel

### Backend
- Spring Boot
- Spring Security
- JWT
- Hibernate
- Render

### Database
- Railway MySQL

### AI
- Google Gemini API
